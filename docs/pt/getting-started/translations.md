# Traduções

O SeminaryERP é multilíngue e, até agora, foi traduzido para português e espanhol.

As traduções são hospedadas e gerenciadas em nosso [projeto no Crowdin](https://crowdin.com/project/seminary-erp).

Ao usar o sistema, fique à vontade para contribuir e melhorar nossas traduções, bem como solicitar um modelo para outros idiomas. Nosso coração é servir a igreja global.

O seminário pode ter um idioma padrão, geralmente definido durante a instalação, e cada usuário pode escolher ver a plataforma no idioma de sua preferência.
